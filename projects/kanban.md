# Kanban — 生活の拠点になるオールインワン・ダッシュボード

> タスク管理（カンバン）、複数AIとのチャット、支出管理、歩数ゲーム、ラジオ再生、カレンダー連携などを1つにまとめた、自分専用の総合ダッシュボードWebアプリ。「毎日ここを開けば生活が回る」を目指したコックピット。

- **リポジトリ**: `Kanban`
- **規模**: TypeScript 2万行以上のフルスタックアプリ

---

## 概要

「カンバン」とは、付箋を貼ったボードのようにタスクを「未着手・今日・明日・今週・完了」の列に並べて管理する方式のこと。このアプリはそのカンバンを中心に、生活に必要な機能をぜんぶ詰め込んだダッシュボード。

特徴は、**Google Tasks（Googleの公式タスク管理）と自動で同期**すること、そして**複数のAIに同じ質問をして答えを比べられる「ディベート機能」**を持つこと。

---

## 主な機能

### カンバンボード（タスク管理）
- Google Tasks と双方向同期（このアプリで作ったタスクがGoogle側にも反映され、逆もまた然り）
- マウスでカードをドラッグして列を移動（直感的な操作）
- 開発タスクは GitHub の Issue（課題管理チケット）と連携

### AIチャット（複数AIの比較）
- **3種類のAI**（Gemini / Grok / GLM-4）に対応。画像を添付して質問もできる
- **ディベートモード**: 2つのAIに同時に質問して回答を左右に並べ、気に入った部分を選ぶと、3つ目のAIがそれらを統合して最適な答えを作る

### その他の生活機能
- **支出管理**: 家計データを月別・週別・カテゴリ別にグラフ表示
- **ラジオ再生**: 自動生成したポッドキャスト（x-yuyaプロジェクト）を再生
- **カレンダー / 日記 / 天気 / RSS（ニュース購読）/ Obsidianメモ閲覧**
- **季節の演出**: 桜・新緑・紅葉・雪のアニメーション
- **朝のあいさつ**: 毎朝7-9時にモチベーションメッセージを表示

## 技術スタック

**フロントエンド（画面側）**
- React 19 + TypeScript（型で安全に書ける言語）+ Vite（高速ビルドツール）
- ドラッグ＆ドロップ（dnd-kit）、グラフ（Recharts）、地図（Leaflet）

**バックエンド（サーバー側）**
- Node.js + TypeScript / Express（定番のWebサーバー）
- Cloudflare Worker（Honoフレームワーク）— ユーザーに近い場所で動く「エッジ」サーバーへ段階移行中

**インフラ（土台）**
- **Cloudflare Pages**: 画面の配信（静的サイトホスティング）
- **Vercel**: APIサーバー（Node.jsライブラリが充実しているため複雑な処理を担当）
- **Cloudflare R2**: 音声ファイルなどの保管庫（オブジェクトストレージ）
- **Cloudflare KV**: 設定や習慣データの軽量データベース

**連携サービス**: Google（Tasks / Calendar / Gemini）、OpenRouter（Grok）、Zhipu AI（GLM-4）、GitHub、Slack、Open-Meteo（天気）、Firebase

## アーキテクチャ

「画面」と「API（処理）」と「保管庫」を分けて、それぞれ得意なクラウドに配置する構成。

```
ユーザー
  │
  ▼
Cloudflare Pages（画面・React）
  │  /api/* のリクエスト
  ▼
Vercel（APIサーバー・28個のエンドポイント）
  ├─ auth        Googleログイン認証
  ├─ kanban      Google Tasks 同期
  ├─ chat/debate/merge  複数AIチャット
  ├─ expenses/steps/badges  支出・歩数
  └─ calendar/obsidian/radio ...
        │
        ├─ Cloudflare Worker（エッジへ移行中の機能）
        ├─ Cloudflare R2（音声）
        └─ Cloudflare KV（設定・習慣）
```

**Vercel と Cloudflare を併用する理由**: Vercelは豊富なライブラリで複雑な処理が書きやすく、Cloudflareはユーザーに近い場所で動くので速くて安い。両方の良いとこ取りをしつつ、複雑な処理を少しずつCloudflareへ移している。

## 技術的な見どころ

- **フルスタックTypeScript（2万行超）**: 画面からサーバー、インフラ設定まで一貫した型安全な開発
- **複数AIの並列制御**: 3つのAIを同時に呼び出し、回答を比較・統合する「ディベート」を実装。回答が少しずつ届くストリーミング表示にも対応
- **段階的なインフラ移行**: 複雑な処理をVercelからCloudflareの「エッジ」へ少しずつ移し、速度とコストを改善。認証・カレンダー・カンバンのコア機能は移行完了
- **Googleとの本格連携**: ログイン認証（OAuth）からタスク・カレンダーの読み書き、トークンの自動更新まで実装
- **自動化**: 毎日の家計集計、型チェック→ビルド→配信の自動デプロイ、AIによるコードレビュー連携
- **コスト最適化**: 各クラウドの無料枠を組み合わせて、ほぼ無料で運用

個人の生産性向上という身近な目的でありながら、認証・エッジコンピューティング・複数AI統合といった、大規模サービスでも使う技術パターンを詰め込んだプロジェクト。
